Cyberduck is a free and open source file transfer client for macOS and Windows. It supports a wide range of protocols — from classic FTP and SFTP to modern cloud storage providers — through a single, consistent interface. A command-line companion tool called `duck` extends the same capabilities to Linux, macOS, and Windows terminals.

## Platforms

Cyberduck runs on **macOS** and **Windows**. The `duck` command-line interface supports **Linux**, **macOS**, and **Windows**, making it suitable for automated transfers and scripting in CI/CD pipelines.

For CLI installation and usage, see [CLI installation](/cli/installation).

## Mountain Duck

The core libraries that power Cyberduck are also used in [Mountain Duck](https://mountainduck.io/), a companion app from the same team that mounts remote servers and cloud storage as a local disk on macOS and Windows.

## Maven artifacts for developers

Cyberduck's protocol implementations are published to a Maven repository on Amazon S3, allowing you to embed them in your own Java projects.

**Repository configuration:**

```xml
<repositories>
  <repository>
    <id>maven.cyberduck.io-release</id>
    <url>https://s3-eu-west-1.amazonaws.com/repo.maven.cyberduck.io/releases</url>
    <layout>default</layout>
    <releases>
      <enabled>true</enabled>
    </releases>
    <snapshots>
      <enabled>false</enabled>
    </snapshots>
  </repository>
</repositories>
```

**Available artifacts:**

```xml
<!-- All protocol implementations -->
<dependency>
  <groupId>ch.cyberduck</groupId>
  <artifactId>protocols</artifactId>
  <type>pom</type>
  <version>7.1.0</version>
</dependency>

<!-- Cocoa Java bindings (macOS only) -->
<dependency>
  <groupId>ch.cyberduck</groupId>
  <artifactId>binding</artifactId>
  <version>7.1.0</version>
</dependency>

<!-- macOS platform implementations (Launch Services, Keychain, etc.) -->
<dependency>
  <groupId>ch.cyberduck</groupId>
  <artifactId>libcore</artifactId>
  <version>${project.version}</version>
</dependency>
```
